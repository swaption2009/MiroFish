# MiroFish Knowledge Vault

This Obsidian vault stores knowledge graphs generated by MiroFish simulation engine.

## Structure

- `_data/graphs/` — Machine-readable JSON graph data (nodes, edges, episodes)
- `Graphs/` — Human-readable Obsidian markdown notes with wiki-links

## Usage

1. Open this folder as an Obsidian vault
2. Browse `Graphs/` to explore extracted entities and relationships
3. Use the Obsidian graph view to visualize connections

## Note

This vault is auto-generated by MiroFish. Manual edits to `_data/` may be overwritten.
